下载安装

Hugo是一个用Go编写的静态站点生成器，由于具有丰富的主题资源和惊人的生成速度而备受青睐。 hugo操作简单，是博客搭建的新手的首选。

hugo的下载安装

  • mac安装只需要在命令行中输入 brew install hugo 再通过hugo version确认是否安装成功

  • windows的下载安装稍微麻烦

    • 在https://github.com/gohugoio/hugo/releases地址下载所需的安装文件包

    根据个人电脑选择对应安装包

    • 系统环境路径的配置 右键的电脑属性 –> 点击高级系统设置 –> 环境变量 –> 选择系统变量Path –> 编辑 –> 新建 –> 将hugo所在的目录添加到新的环境变量
      环境变量添加完成后在系统命令行中即可运行hugo指令

git下载安装

git的下载地址为https://git-scm.com安装过程非常简单
系统环境路径的配置也和hugo配置相同

使用hugo创建第一个博客

在系统命令行中输入hugo new site test–自拟博客文件名称 博客创建成功输入dir查看test博客文件目录

C:\Users\administrator>hugo new site test

为博客文件下载主题

此时创建的博客为空文件夹，还需要为博客下载主题 https://themes.gohugo.io/为hugo主题网站选择想要的主题，并前往github下载主题，github上readme.md文件中会如下所示的下载路径复制

git clone https://github.com/zhaohuabing/

到命令行即可下载主题，注意先将当前目录调整到themes文件。此时themes文件中就有下载好的主题文件夹

写一篇博客

使用在博客目录下使用hugo new post/firstarticle.md在post文件夹下创建md一篇文件，使用visual studio code编写博客

博客发布–通过github

申请respository

首先在自己的github账号中申请一个repository名字要和自己的账号名保持一致加上 .github.io

为本地博客绑定链接

hugo –theme=hugo-theme-cleanwhite –baseUrl=“https://liusheng996.github.io" –buildDrafts
博客文件夹会出现一个public文件为将来要发布的文件

public文件传到本地git仓库中

git init
命令行在public目录下输入git init将public变成Git可以管理的仓库 命令行输入完成瞬间git就把仓库建好了 public目录下会有.git目录出现这个目录是Git来跟踪管理的版本库，一般不要乱动.git目录 以免Git库被破坏
git add
将文件添加到仓库 git add .表示将public中的文件全部加入到仓库中 git commit 将文件添加到仓库 git commit -m “提交说明”

要把本地git库和远程github进行关联

git remote add origin http://github.con/liusheng996/liusheng996.github.io.git

将文件push到github库中

git push -u origin master

完成

访问http://liusheng996.github.io

个人总结：

hugo的主题选择和调整真的是件麻烦事，个人经验刚开始用千万不要使用一些不太成熟的主题 这些主题中没有example site文件 学习调试起来非常懵逼 直接选择官网上比价成熟的项目有利于学习理解

这里推荐bilibili上的hugo调试的教学视频https://www.bilibili.com/video/av58083222

主题的调试是件很麻烦的事，网上相关资料也很缺乏，后期会总结个人经验

这里还要建议使用给git搭建个人博客要掌握一定的git知识 推荐廖雪峰的git教程
